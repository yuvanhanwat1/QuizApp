# QuizApp
Quiz App Using React !

## 5 Steps to Run this Project 
1. Select any folder and open the terminal of your IDE.
2. Install the react app using this command : 
> npx create-react-app QuizApp
3. After creating the react app only change the src folder.
4. After creation, your project should look like this:

* QuizApp/
    - node_modules/
    - public/
    - src/
        - App.js
        - index.css
        - index.js
        - components/
            - QuestionBox.js
            - ResultBox.js
        - Data/
            - index.js
          
* For the project to build, these files must exist with exact filenames:      
    - public/index.html is the page template;
    - src/index.js is the JavaScript entry point.

5. the type this command to run the app :
> npm start
- Runs the app in the development mode.
- Open http://localhost:3000 to view it in the browser.

- The page will reload if you make edits.
- You will also see any lint errors in the console.
